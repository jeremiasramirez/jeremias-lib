  • (0) minmin: size very min
  • (1) min: size min-small

  • (3) medium: size medium
  • (4) large: size large 100%

    • MENSAJES DE ALERTAS css

  • (1) alert--error: muestra un mensaje de 'error' al usuario
  • (2) alert--success: para introducir un mensaje de 'error' al usuario
  • (3) alert--warning: para introducir un mensaje de 'advertencia' al usuario
  • (4) alert--chargin: para introducir un mensaje de 'carga de datos' al usuario

    • BOTONES css

  • (1) btn--add: boton con sintaxis de 'aceptar condicion'
  • (1) btn--yes: boton con sintaxis de 'agregar elementos'
  • (2) btn--rm: boton con sintaxis de 'remover elemento o cancelar operacion'
  • (2) btn--info: boton para mostrar o enviar informacion'

    • FORMULARIOS css

  • (1) form: clase que le da una buena sintaxis al formulario, removiendo curvaturas no agradables.
  • (2) check: clase para label que contenga inputs con type=checkbox
  • (2) select: estiliza la etiqueta select

    • ELEMENTOS VISUALES

  • (1) close: elemento que elimina al elemento padre del mismo (ojo, el padre debe tener el Id "parent").
  • (2) blur: clase que le da una opacidad a cierto elemento.
  • (3) dark: clase que le da un fondo negro a un elementos.
